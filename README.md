# 🛒 Blinkit Clone

A responsive **Blinkit Landing Page Clone** built using **HTML5** and **CSS3**. This project recreates the look and feel of Blinkit's homepage while focusing on responsive web design, clean UI, and modern CSS techniques.

## 🚀 Live Demo

🔗 https://jayeshbansal.github.io/Blinkit-Clone/

## ✨ Features

- Responsive design for Desktop, Tablet, and Mobile
- Modern Blinkit-inspired UI
- Navigation bar with search section
- Hero banner section
- Product category cards
- Promotional banners
- Product showcase sections
- Footer with useful links
- Clean and organized code
- Cross-browser compatible

## 🛠️ Built With

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Media Queries

### Navigate to the Project

```bash

```

Simply open `index.html` in your preferred web browser.
